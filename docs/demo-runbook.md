# VIP Live Demo Runbook

## Before the pitch

1. Run `npm run dev`.
2. Open the local URL shown in the terminal.
3. Keep this runbook open in a separate window.
4. If the dashboard contains old data, open Dashboard and click Clear local data.

## Case 1 - Hidden workload becomes visible

1. Click Demo Cases.
2. Click Run demo on Case 1 - Hidden workload becomes visible.
3. The app opens Dashboard automatically.
4. Point to the Deadline Bank and say that tasks come from several courses.
5. Point to Week 3 in the heatmap.
6. Point to Academic Load Score and read the four transparent categories.
7. Point to Action Card and say that every heavy week leads to a next step.

## Guided week tour

1. Open Dashboard.
2. Click Start tour in Guided week tour.
3. Let the tour move through Week 1, Week 2, Week 3, and Week 5.
4. Point out that the heatmap, Academic Load Score, Action Card, and Deadline Bank follow the selected week automatically.
5. Pause on Week 3 and say: "This is where hidden workload becomes visible before the week starts."

## Case 2 - Add Task live demo

1. Click Demo Cases.
2. Click Run demo on Case 2 - Add Task live demo.
3. The app opens Dashboard automatically with the baseline workload and a Project Milestone prefilled in the Add task form.
4. Click Add task.
5. Point to Week 3 in the heatmap and show that it is Red.
6. Point to Deadline Clustering and Expected Effort in the score panel.
7. Point to the Red Week Action Card.
8. Say: "The value is not only the warning. The value is the next 24-48 hour plan."

## Optional manual Add Task version

Use this if judges ask to see live input.

1. Open Dashboard.
2. In Add task, enter:
   - Course: Foundations Project
   - Task: Project Milestone
   - Due date: 2026-09-24
   - Task type: Large
   - Expected effort: High
   - Clarity: Clear
   - Weight: High
3. Click Add task.
4. Point to the heatmap, score, and action card changing immediately.

## Case 3 - Unclear assignment and support route

1. Click Demo Cases.
2. Click Run demo on Case 3 - Unclear assignment and support route.
3. The app opens Dashboard.
4. Show Writing Reflection Essay marked Unclear in the Deadline Bank.
5. Point to Task Clarity Risk in the score panel.
6. Click Open SOS Navigator on the Action Card.
7. Select Assignment unclear.
8. Show the route: TA / Instructor.
9. Scroll to the clarification email template.
10. Say: "VIP frames support as where to ask, not what is wrong with the student."

## Closing screen

1. Open Pilot & Metrics.
2. End with the pilot ask:
   - Permission to pilot
   - Connection with Peer Advisors, SAM, and support teams
   - Recruitment support for first-year volunteers
